# LinkedIn-Sentiment-Keyword-Analyzer
## The Mission
Instead of just scrolling, I wanted to understand the "pulse" of my professional network. This Python tool analyzes LinkedIn comments to determine sentiment and visualize the most discussed topics.

## 🛠️ Tech Stack
- **Python 3.x**
- **TextBlob:** For Natural Language Processing (NLP) and sentiment grading.
- **WordCloud & Matplotlib:** For data visualization.

## 🚀 How to Use
1. Clone the repo.
2. Add your LinkedIn comment strings to the `comments` list.
3. Run `python linkedin_analyzer.py`.

## 📈 Impact
This project moves beyond basic syntax to demonstrate how data science can be applied to social media strategy and community engagement.
